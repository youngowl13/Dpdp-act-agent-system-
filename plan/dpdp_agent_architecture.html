DPDP Act Core Principles

Purpose Discipline
Personal data used exclusively for clear, legitimate purposes
Data Minimization
Only collect data strictly required for stated purpose
Retention Limitation
Automated erasure once purpose is fulfilled
Accuracy
Ensure accuracy in data-driven decisions
Security Safeguards
Reasonable protection against unauthorized access
Transparency
Clear and transparent processing operations
Accountability
Demonstrable audit trails for all actions
